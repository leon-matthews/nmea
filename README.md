
# NMEA Parser

Parse various types of NMEA sentences from GPS receivers.

As much an exercise in using Python's new dataclasses as anything else.

## Devices

In ordinary terminal::

    $ stty -F /dev/ttyUSB0 4800
    $ cat /dev/ttyUSB0
